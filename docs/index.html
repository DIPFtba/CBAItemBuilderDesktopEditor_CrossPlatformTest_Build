Installation

macOS

  • Download and extract the .tar.gz file
  • Move cba-itembuilder-10.4.0.app to Applications
  • Before first launch, open Terminal and run:
xattr -cr /Applications/cba-itembuilder-10.4.0.app

This removes the macOS quarantine flag (required because the app is not code-signed).

Linux

  • Extract: tar xzf cba-itembuilder-10.4.0-linux.gtk.x86_64.tar.gz
  • Run: ./cba-itembuilder-10.4.0/cba-itembuilder
  • Requires libwebkit2gtk-4.0 for the preview feature

Windows

  • Extract the .zip file to a location of your choice
  • Run cba-itembuilder.exe from the extracted folder
Note: All downloads include an embedded Java Runtime (Temurin JRE 11). No separate Java installation is required.
